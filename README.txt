梅田校 夏期講座スケジューラー v5

使い方:
1. index.html をブラウザで開く
2. 「スケジュール作成設定」で優先する先生を選ぶ
3. 講座一覧を「夏期特訓」「夏期講習」→「教科」ごとに展開する
4. 受けたい講座の「候補」を押す
5. 「選択講座から自動作成」を押す
6. 重複・校舎移動の警告を確認する

主な機能:
- 特訓・講習ごとの折りたたみ表示
- 各区分内の教科ごとの折りたたみ表示
- 科目・校舎・区分・キーワード検索
- 優先講師を検索条件ではなく、日程作成用の固定設定として管理
- 優先講師の複数選択とブラウザ保存
- 候補講座からおすすめ日程を自動算出
- 2限と3限の間のみ校舎移動可能として判定
- 重複・移動不可の警告表示
- CSV出力

更新 v5:
- 優先する先生を検索欄から分離し、「スケジュール作成設定」に移動。
- 優先講師は講座一覧の絞り込みには使わず、自動作成時の評価に一貫して反映。
- 選択した優先講師を localStorage に保存し、再読み込み後も維持。
- 優先講師に該当する講座を一覧・予定上で視覚的に表示。

注意:
PDFから抽出した講座表をもとにした静的サイトです。実際の申込前には、最新の公式時間割で確認してください。
