梅田校 夏期講座スケジューラー v3

使い方:
1. index.html をブラウザで開く
2. 講座一覧を「夏期特訓」「夏期講習」→「教科」ごとに展開する
3. 受けたい講座の「候補」を押す
4. 必要に応じて「優先する先生」を入力する
5. 「選択講座から自動作成」を押す
6. 重複・校舎移動の警告を確認する

主な機能:
- 特訓・講習ごとの折りたたみ表示
- 各区分内の教科ごとの折りたたみ表示
- 科目・校舎・区分・キーワード検索
- 候補講座からおすすめ日程を自動算出
- 優先講師の入力とスコア反映
- 2限と3限の間のみ校舎移動可能として判定
- 重複・移動不可の警告表示
- CSV出力

注意:
PDFから抽出した講座表をもとにした静的サイトです。実際の申込前には、最新の公式時間割で確認してください。
