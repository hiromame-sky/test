梅田校 夏期講座スケジューラー v7

このサイトは、アップロードされた2026年度 梅田校 夏期講座PDFをもとにした静的HTMLサイトです。

v7の主な変更点
- 講座名だけを候補に入れる方式へ変更
- 候補追加時に校舎・組・講師を固定しない仕様へ変更
- 同じ講座名に複数の校舎・組・講師・日程がある場合、自動作成時に最適な候補を選択
- 優先する先生を共通設定として導入
- 優先する校舎も共通設定として導入
- 共通設定は検索や候補追加には使わず、自動作成時の評価にのみ反映
- 2限と3限の間のみ校舎移動可能として判定
- 夏期特訓・夏期講習、その中の教科ごとの折りたたみ表示を維持

使い方
1. index.htmlをブラウザで開く
2. 講座名一覧から受けたい講座名を「候補に入れる」
3. 共通設定で優先する先生や校舎を選ぶ
4. 「講座名候補から自動作成」を押す
5. 重複や校舎移動条件を確認する
6. 必要に応じてCSV出力する

注意
このサイトはPDFから抽出したデータをもとにしています。実際の申込前には、必ず公式の最新時間割で確認してください。
