梅田校 夏期講座スケジューラー v8

使い方
1. index.html をブラウザで開きます。
2. 講座名一覧から、受けたい講座名だけを候補に入れます。
3. 共通設定で、優先する先生・優先する校舎・映像講座利用・日程圧縮を設定します。
4. 「講座名候補から自動作成」を押すと、校舎・組・講師・日程を自動で選びます。
5. 完成した予定はCSVで出力できます。

v8の修正点
- 講座名一覧が表示されない場合がある問題を修正しました。
- 講座名の取得を course / name / title / 講座名 の複数キーに対応させ、データ形式の揺れに強くしました。
- 候補管理を講座名文字列だけではなく、区分・教科・講座名を含む内部IDで管理するように変更しました。
- 同じ講座名が別区分・別教科に存在しても、一覧や候補から消えにくいようにしました。
- 特訓・講習、その中の教科アコーディオンは初期状態で展開し、講座名が見える状態にしました。
- 検索対象に講座名だけでなく、教科・区分・校舎・講師も含めました。

判定仕様
- 同じ時間帯に重なる講座は重複として扱います。
- 校舎移動は2限と3限の間のみ可能として扱います。
- 優先講師・優先校舎は検索条件ではなく、自動作成時のスコアにだけ反映します。
